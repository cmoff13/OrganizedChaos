I choose to learn web development because:

  • I aspire to become a well-rounded graphic designer
  • There is a big need for it today in the workforce
  • I must know how to code in order to understand build and wireframe websites
  • It gives me an advantage over someone who doesn't know web development in the design industry
  • I figure it would be a good thing to know for future employment
I am most excited to learn...
Javascript

I am excited to learn Javascript because I enjoy interactive web pages and how they work. I have never dealt with animation and motion graphics, let alone coding in general, so I hope it will come easy to me. I also am stoked to learn Javascript because all the big jobs hiring right now require knowledge of coding with Javascript and it will be big advantage for my resume.
